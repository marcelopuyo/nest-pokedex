## Ejecutar en desarrollo

1. Ejecutar en desarrollo
2. Ejecutar:
```bash
npm install
```
3. Tener Nest CLI instalado
```bash
npm install -g @nestjs/cli
```
4. Levantar la base de datos
```bash
docker-compose up -d
```

* Nest
